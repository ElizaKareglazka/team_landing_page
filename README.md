# team_landing_page
Одностраничный сайт-визитка команды разработки X5 Tech. Выполняет функцию электронной визитки: рассказывает о команде, её участниках, проектах, процессе работы и текущем roadmap. Даёт внешним и внутренним заказчикам понимание, чем занимается команда и с какой задачей к ней можно обратиться.
